# Pro First Aid App

A professional, real-time first aid guide for Android, designed to provide quick and reliable medical instructions during emergencies.

## Features

- **Quick Emergency Access**: A dedicated Floating Action Button (FAB) to immediately dial emergency services (911).
- **Comprehensive Guides**: Detailed, step-by-step instructions for life-threatening and common injury scenarios:
    - CPR (Cardiopulmonary Resuscitation)
    - Choking (Heimlich Maneuver)
    - Severe Bleeding Control
    - Burn Treatment
- **Smart Search**: Real-time filtering to find specific topics or categories (e.g., "Life Threatening") instantly.
- **Modern UI/UX**: 
    - Material 3 Design components.
    - Collapsing Toolbar with parallax headers.
    - Clean, card-based navigation.
    - Numbered, easy-to-read instruction steps.

## Tech Stack

- **Language**: Java
- **UI Framework**: Android XML with Material Components
- **Architecture**: Clean, modular data handling using `TopicData` and `TopicAdapter`.
- **Target SDK**: 36 (Android 15+)
- **Min SDK**: 24 (Android 7.0+)

## Installation

1. Clone or download the project.
2. Open in **Android Studio**.
3. Sync Gradle and build the project.
4. Run on an emulator or a physical device.
